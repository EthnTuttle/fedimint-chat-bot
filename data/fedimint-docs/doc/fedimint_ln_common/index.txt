LIGHTNING MODULE

This module allows to atomically and trustlessly (in the federated trust model) interact with the Lightning network through a
Lightning gateway.


ATTENTION: ONLY ONE OPERATION PER CONTRACT AND ROUND

If this module is active the consensus’ conflict filter must ensure that at most one operation (spend, funding) happens per
contract per round
