Fedimint Gateway Binary

This binary runs a webserver with an API that can be used by Fedimint clients to request routing of payments through the Lightning
Network. It uses a GatewayLightningClient, an rpc client to communicate with a remote Lightning node accessible through a
GatewayLightningServer.
